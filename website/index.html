Before you download
macOS will warn you — that's normal

Auchive is safe. macOS shows a warning because it requires developers to pay Apple $99/year for a verified certificate. Auchive is an independent app and doesn't have one yet. Here's how to open it:

  1. Open the .dmg file and drag Auchive to your Applications folder
  2. Try to open Auchive — macOS will block it with a warning
  3. Go to System Settings → Privacy & Security and scroll down
  4. Click "Open Anyway" next to the Auchive message

You only need to do this once. After that, Auchive opens normally.
